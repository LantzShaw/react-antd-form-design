# React Antd Form Design

基于 React 和 Antd 的表单设计器，支持表单渲染、表单校验、字段分组等功能。

### 支持以下功能

- [ ] 表单设计器
- [ ] 表单渲染
- [ ] 表单校验
- [ ] 套件化配置
- [ ] 字段分组
- [ ] 自定义组件

### 内置组件

- [x] Input
- [x] Radio
- [x] Checkbox
- [x] Select
- [x] DatePicker
- [x] TimePicker
- [x] InputNumber
- [x] Switch
- [x] Slider
- [x] Rate
- [x] Upload
- [x] Cascader
- [x] TreeSelect
